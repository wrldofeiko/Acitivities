Website Analysis: Arngren.net

User Experience

The UX is very cluttered. There is too much visual noise from moving images, making it hard for a user to focus on a single task or product.

Content

The site contains a vast amount of product information, but it lacks a search function or clear categorization, making the content difficult to consume.

Grid Layout

The layout does not follow modern grid principles. It appears to be an unorganized collection of frames and tables that are not mobile-responsive.
